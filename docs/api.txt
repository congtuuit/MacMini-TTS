### 1. Lấy danh sách giọng mặc định
curl --location 'http://localhost:8000/api/voices' \
--header 'accept: application/json'

---

### 2. Sinh giọng nói (TTS)
# Bạn có thể truyền "voice" là tên giọng đã lưu hoặc câu lệnh mô tả giọng (vd: "female, high pitch").
# Các tham số mở rộng (chỉ dành cho OmniVoice instruct):
# - seed (int): Cố định seed ngẫu nhiên để sinh ra các giọng khác nhau.
# - keep_voice (bool): Mặc định false. Nếu đặt là true, hệ thống sẽ tự động cache chất giọng sinh ra ở lần gọi đầu tiên và tái sử dụng làm file mẫu (clone) cho các lần sau, giúp giọng ổn định 100% dù text thay đổi.
curl --location 'http://localhost:8000/api/tts' \
--header 'accept: application/json' \
--header 'Content-Type: application/json' \
-s --dump-header - \
--output test_voice.wav \
--data '{
  "text": "Xin chào, tôi là AI hỗ trợ đọc tiếng Việt. Nghe hay quá đi [cười].",
  "voice": "female, high pitch",
  "seed": 123,
  "keep_voice": true
}'

---

### 3. Clone giọng nói (Zero-shot Voice Cloning)
# Sử dụng form-data (-F) để upload file âm thanh mẫu (ref_audio).
# Chuẩn bị sẵn 1 file mau_giong.wav (khoảng 3-5 giây) ở thư mục hiện tại để test.
curl --location 'http://localhost:8000/api/clone' \
--header 'accept: application/json' \
-s --dump-header - \
--output cloned_voice.wav \
-F 'text="Đây là câu nói được clone trực tiếp từ file âm thanh mẫu tôi vừa gửi lên."' \
-F 'ref_audio=@mau_giong.wav'
